1. System Architecture Overview & Analysis

This report documents the full codebase analysis of the monolithic MVC project and the completed decoupled Angular SPA & ASP.NET Core Web API architecture.

Key Architectural Differences

Component Monolithic MVC Solution Decoupled Web API + SPA
Frontend Rendering Razor Views (.cshtml) Server Side Angular SPA Client Side
Data Access DbHelper.cs (ADO.NET) Raw SQL EF Core (Fluent API & LINQ)
Auth & Session Cookie Session (Stateful) JWT Bearer Tokens (Stateless)
Real-time SignalR ChatHub (Cookie) SignalR ChatHub (Token query string)

Clean Architecture Directory breakdown

  • Portfolio.Domain: Isolated project containing Entity model definitions.
  • Portfolio.Application: Declares services, DTOs, and interface boundaries.
  • Portfolio.Infrastructure: Relational Entity Framework DB Context, Token Service, and Email Service.
  • Portfolio.WebAPI: REST-based controller entrypoints, middleware, CORS, and auth hooks.
  • Portfolio.SPA: Angular frontend app routing, views, widgets, and interceptors.
